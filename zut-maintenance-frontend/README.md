# ZUT Maintenance Reporter Frontend

React frontend for ZUT maintenance reporting and administration.

## Getting Started

1. Run `npm install`
2. Run `npm run dev`

## Backend API

This app expects the backend API at `http://localhost:5000/api`.

## Available Scripts

- `npm run dev` — start the Vite development server
- `npm run build` — build the production bundle
- `npm run preview` — preview the production build locally
